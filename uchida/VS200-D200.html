Uchida VS-200/D-200

Ver Ficha Técnica

Las perforadoras UCHIDA ofrecen la opcion mas amplia de tamaño, capacidad y sofisiticacion. De facil uso con tablas corredizas a cada lado o fijadores a cada lado. Todos los modelos referenciados cumples con las pautas HSE. Adecuado para papeleria convencional y de computador. Son ideales para departamentos reprograficos, impresores, archivadores y oficinas. La UCHIDA VS-200 es un taladro sicnronizado de doble eje con una barra de perforado pre programada, pila de 50mm de altura para papeles de hasta A3, banco montado con guias centricas para precision.

El D-200 es una cabeza de perforado con una barra de de perforado universal, funciona con pilas de papel de hasta 50mm en tamaños de hasta A3.
Taladro sincronizado de doble eje (VS-200), cabezas fijas de un lado de perforado (D-200)
